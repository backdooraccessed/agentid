Installation

1. Add the script

<script src="https://agentid.dev/widget.js"></script>

2. Add a widget container

<div
  data-agentid-widget
  data-credential-id="your-credential-id"
  data-theme="light"
></div>

3. Or use JavaScript API

AgentID.widget({
  container: '#my-container',
  credentialId: 'your-credential-id',
  theme: 'dark',
  compact: false,
  onVerified: (result) => {
    console.log('Verified:', result);
  },
  onError: (error) => {
    console.error('Error:', error);
  }
});

Data Attributes

Attribute Description
data-credential-id The credential ID to verify (required)
data-theme "light" or "dark" (default: light)
data-compact Enable compact mode (no value needed)
data-hide-score Hide the trust score badge
